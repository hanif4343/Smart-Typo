এটাই তোমার আসল "পারমিশন" — এখন Firestore এর `users/{uid}` ডকুমেন্টের `aiEnabled` ফিল্ড দিয়ে নিয়ন্ত্রিত হয়। বদলাতে হলে Firebase Console → Firestore Database → Data ট্যাবে গিয়ে নিজের user doc খুঁজে `aiEnabled` কে `true` করে দাও। সাইন-ইন করা না থাকলে AI সবসময় বন্ধ থাকবে।

এই key শুধু তোমার ব্রাউজার/অ্যাপের local storage-এ থাকে। কোনো সার্ভারে বা GitHub repo-তে যায় না। App uninstall/browser data clear করলে আবার বসাতে হবে।
